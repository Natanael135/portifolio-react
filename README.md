## 🚀 Como Executar

1. **Clone o repositório**
   ```bash
   Git clone https://github.com/Natanael135/
   ```
2. **Instale as dependências**
   ```bash
   npm install
   ```
3. **Execute o servidor de desenvolvimento**
   ```bash
   npm run dev
   ```
4. **Acesse no navegador**
   ```bash
   http://localhost:5173
   ```
